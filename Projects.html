University Course Manager

This was a project based on handling a student's university courses data efficiently.

I have used the concept of Data File Handling in C++ to make this project. The features of this project are:

  • Add one or more courses that is append.
  • Modifying the course details of a course which is already entered by the student.
  • Searching a specific course's detail entered by the student.
  • Deleting a course entered by the user
  • Displaying all the courses taken by a student
